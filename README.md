# Love2D Arkanoid
Pequeño proyecto para probar el framework Love2D https://love2d.org/

También para probar mi primer contacto con LUA.

## Ejecutar
Descarga o instala Love y ejecuta el comando ```love .``` en el directorio raíz.

## Jugar
- Puedes mover la pala a izquierda y derecha con las teclas de dirección o con las teclas A y D.
- Pantalla completa, tecla F
- Para cerrar el juego, tecla ESC
- Reglas de Arkanoid (sin vidas ni puntuación, juego muy sencillo)
